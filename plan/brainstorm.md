# 01 - Brainstorm

## Target Audience
**Primary:** Recruiter dan Hiring Manager di bidang IT (Web Development, Technical Support, IT Infrastructure)

**Secondary:** Potential clients untuk freelance web development projects

## Portfolio Goals
1. Menunjukkan technical skills (Web Development, UI/UX, Database, System Administration)
2. Membuktikan kemampuan problem-solving melalui projects
3. Menampilkan profesionalisme sebagai fresh graduate yang siap kerja
4. Membuat first impression yang kuat dan memorable

## Key Sections Needed

### 1. Hero Section
- Tagline profesional yang menjelaskan value proposition
- Call-to-action jelas (Download CV, Contact)
- Photo/avatar profesional

### 2. About Me
- Short bio (2-3 paragraphs)
- Highlight: Fresh graduate + ready to contribute
- Soft skills: problem-solving, fast learner, team player

### 3. Skills Section
**Technical Skills:**
- Frontend: HTML, CSS, JavaScript
- Backend: PHP, MySQL
- Design: UI/UX Design principles, Figma
- Systems: Windows Installation & Configuration, Troubleshooting

**Soft Skills:**
- Problem Solving
- Team Collaboration
- Fast Learner
- Attention to Detail

### 4. Projects/Portfolio
Minimum 3 projects yang showcase different skills:
1. **Web Development Project** - Full-stack website dengan database
2. **UI/UX Design Project** - Design case study atau redesign
3. **Technical Project** - System installation/configuration documentation

### 5. Education
- Degree/program
- Relevant coursework
- Academic achievements (jika ada)

### 6. Contact Section
- Email
- LinkedIn
- GitHub
- WhatsApp/Phone (optional)

## Design Direction

### Color Scheme
**Professional & Trustworthy:**
- Primary: Navy Blue (#1e3a8a) - trust, professionalism
- Secondary: Slate Gray (#64748b) - modern, technical
- Accent: Emerald (#10b981) - growth, fresh perspective
- Background: White/Light Gray (#f8fafc)

### Typography
- Headings: Sans-serif modern (Inter, Poppins)
- Body: Readable sans-serif (System fonts)
- Hierarchy jelas: H1 → H2 → H3 → body

### Layout Principles
1. **Mobile-First:** Semua section harus mobile-friendly
2. **White Space:** Generous spacing untuk readability
3. **Visual Hierarchy:** Important info stand out
4. **Consistency:** Spacing, colors, typography consistent

## Content Tone

### Voice Characteristics
- **Professional** tapi tidak kaku
- **Confident** tapi tidak arrogant
- **Clear & Direct** - no fluff
- **Action-oriented** - fokus pada achievements

### Example Phrases
❌ "I think I'm quite good at web development"
✅ "Experienced in building responsive web applications with PHP and MySQL"

❌ "I hope to get a job in IT"
✅ "Seeking opportunities to apply technical skills in web development and IT infrastructure"

## Competitive Analysis

### What Makes This Portfolio Stand Out
1. **Diverse Skill Set:** Bukan cuma coding, tapi juga design dan system administration
2. **Documentation Quality:** Menunjukkan kemampuan technical writing
3. **Clean Implementation:** Code yang rapi, semantic HTML, organized CSS
4. **Attention to Detail:** Responsive, accessible, professional

## User Journey

### First Visit (5-10 seconds)
1. User lands on hero → sees name + title
2. Reads tagline → understands value proposition
3. Sees CTA → knows next action

### Engaged Visit (30-60 seconds)
1. Scrolls to Skills → validates technical competency
2. Checks Projects → sees proof of skills
3. Reads About → connects with personality
4. Decides to contact or download CV

### Recruiter Visit (2-3 minutes)
1. Quick scan of all sections
2. Deep dive into 1-2 relevant projects
3. Check GitHub/LinkedIn for validation
4. Save contact or send interview invite

## Success Metrics
- Portfolio loads in < 2 seconds
- Mobile responsive (works on 320px - 1920px)
- All code understandable (can explain to recruiter)
- Professional impression in first 5 seconds
- Clear path to contact/next steps
